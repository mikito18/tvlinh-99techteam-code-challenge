# Token Swap App

A token swap application built with React, TypeScript, and Vite.

## System Requirements

- **Node.js**: >= 20.0.0
- **Yarn**: >= 4.0.0 (recommended)

## Installation

**Install dependencies:**

```bash
yarn install
```

## Running the Application

### Development Mode

```bash
yarn dev
```

The application will run at: http://localhost:3000

### Build for Production

```bash
yarn build
```

### Preview Build

```bash
yarn preview
```

### Code Linting

```bash
yarn lint
```
